Color helpers allow you to modify the colors of HTML elements via CSS class.

## Background colors

The following helpers, starting with `.ms-bg-***`, allow you to modify the background of an HTML element.

```html live
<div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-main">.ms-bg-main</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-light">.ms-bg-light</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-primary">.ms-bg-primary</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-secondary">.ms-bg-secondary</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-action">.ms-bg-action</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-action2">.ms-bg-action2</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-white">.ms-bg-white</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-black ms-text-white">.ms-bg-black</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-yellow">.ms-bg-yellow</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-green">.ms-bg-green</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-red">.ms-bg-red</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-purple">.ms-bg-purple</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-orange">.ms-bg-orange</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-brown">.ms-bg-brown</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-navy ms-text-white">.ms-bg-navy</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-bluesky">.ms-bg-bluesky</div>
    <div class="ms-alert ms-border ms-fullwidth ms-bg-pink">.ms-bg-pink</div>
</div>
```

## Invert color

It is possible to reverse the colors with the `.ms-invert-color` class.

```html live
<div class="ms-alert ms-fullwidth ms-primary ms-invert-color">.ms-invert-color</div>
```
