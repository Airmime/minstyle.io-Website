The position of the blocks can be changed with the class `ms-display-***`.

```html
<div class="ms-display-none">display: none;</div>
<div class="ms-display-block">display: block;</div>
<div class="ms-display-inline">display: inline;</div>
<div class="ms-display-inline-block">inline-block;</div>
<div class="ms-display-flex">display: flex;</div>
```
