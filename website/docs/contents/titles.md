By default, minstyle.io defines a size for title tags like `h1` or `h2`.

```html live
<div>
    <h1>H1 Title - 3.5rem</h1>
    <h2>H2 Title - 3rem</h2>
    <h3>H3 Title - 2.5rem</h3>
    <h4>H4 Title - 2rem</h4>
    <h5>H5 Title - 1.5rem</h5>
    <h6>H6 Title - 1.25rem</h6>
</div>
```
